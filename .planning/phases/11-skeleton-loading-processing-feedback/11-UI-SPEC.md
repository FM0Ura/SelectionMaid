## Interaction & Animation Specs

### Layout Transition: "Minimize Card"
- **Trigger**: `useUpload` status transitions to `processing`.
- **Motion**:
  - `DropZone` height reduces from `min-h-96` to `min-h-48`.
  - Padding reduces from `p-8` to `p-6`.
  - Content shifts from central vertical stack to horizontal/compact layout.
  - Skeletons animate in below using `initial: { opacity: 0, y: 20 }, animate: { opacity: 1, y: 0 }`.

### Shimmer Animation
- **Effect**: Linear sweeping gradient.
- **Colors**: `oklch(0.205 0 0)` (base) -> `oklch(0.922 0 0)` (shimmer) -> `oklch(0.205 0 0)` (base).
- **Timing**: 2s duration, linear easing, infinite loop.
- **Angle**: 90deg (left to right).

### Processing Timer
- **Focal Point**: Primary focal point during processing state.
- **Start**: Immediately on `processing` status.
- **Interval**: 1000ms.
- **Location**: Prominent, central within the minimized card.
